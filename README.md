## セットアップ方法
1. Unity Hubをインストールしてください。
2. Unityのバージョンは **2021.3.x LTS** を使用してください（推奨）。
3. このリポジトリをクローンします：
    ```bas
    hgit clone <リポジトリURL>
4. Unity Hubでプロジェクトを開きます。
